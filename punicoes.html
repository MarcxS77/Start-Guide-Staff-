Aplicando uma punição

O primeiro passo para aplicar uma punição é verificar se o jogador já está punido pelo motivo que você deseja punir, pelo comando /checkpunir
Após a verificação, será necessário digitar o comando /punir .

Ao digitar o comando, será aberto este menu:
Cada punição tem ID próprio e requer que as provas sejam enviadas no chat.

Basta selecionar a punição desejada e enviar as provas no chat.

Abra em uma nova guia para ver melhor.

Verificando o histórico de punições de um jogador

Para verificar o histórico de um jogador, basta digitar o comando /checkpunir .

Revogando uma punição

Para revogar uma punição, basta digitar o comando /revoke . O ID pode ser obtido pelo histórico de punições do jogador. Após executar o comando, essa mensagem será enviada:

Depois, basta selecionar o motivo desejado e confirmar sua ação.
